Logging In & Session Management

Visit the secure login page and enter your assigned AccSafety credentials. The portal displays a mandatory data use acknowledgement before you can proceed.

  • Remembered access: After accepting the policy notice once per browser, you can sign in directly until the consent is reset.
  • Session timeout: Select Log out in the header when you are finished or need to switch accounts.
  • Forgot credentials? Contact the program administrator to reset your password—self-service reset is not yet available.

Using the Dataset Explorer

Select Explore Available Datasets from the portal home to launch the unified explorer. Use the toolbar at the top of the explorer to refine data:

Mode
Filter to pedestrian, bicycle, or combined multimodal counts.
Facility
Highlight on-street lanes, trails, bridges, or other facility types to tailor results.
Data Source
Toggle between automated counters, short-duration studies, legacy sources, and live feeds.

Hover map features and charts for contextual insights. Each dataset card includes key metrics, download links, and shortcuts to related applications.

Exploring Specialized Applications

Use the quick links below to jump into focused workflows. Each application opens in a protected route—bookmarking is safe once you are signed in.

  • Trail Counters Dashboard – Analyze permanent and seasonal trail counts, compare corridors, and export chart data.
  • Eco-Counter Explorer – Review Eco-Counter device performance, health alerts, and supporting maintenance logs.
  • Vivacity Analytics – Monitor computer-vision detections and safety insights around high-priority intersections.
  • Live Detection Monitor – Watch real-time multimodal activity and verify sensor uptime.

Need Help?

Email accsafety-support@example.com with questions about data updates, account provisioning, or to request onboarding materials for new team members.

For urgent system outages, contact the program lead by phone using the escalation tree shared in onboarding materials.
